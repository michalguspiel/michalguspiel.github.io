Further plans


Learning Flutter

Flutter is an open-source UI software development kit created by Google. It is used to develop cross platform applications for Android, iOS, Linux, Mac, Windows, Google Fuchsia and the web from a single codebase.

It is a very interesting technology worth learning. Strong knowledge of Android development as well as Kotlin would be very helpful because it is developed by Google and Flutter apps are written in Dart which is really similar to Kotlin.


Or React Native

React Native is a JavaScript framework for writing real, natively rendering mobile applications for iOS and Android. It’s based on React, Facebook’s JavaScript library for building user interfaces, but instead of targeting the browser, it targets mobile platforms.

Knowledge of React Native is really beneficial for a mobile developer since a lot of cross platform apps are built with this framework.
